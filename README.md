scoobi-playground
=================

Scoobi Playground

The repository contains some examples to get started with Scoobi. 

To start:
* 1. Install sbt (http://www.scala-sbt.org/)
* 2. Clone https://github.com/anwarrizal/scoobi-playground.git
* 3. Open file WordCount.scala
* 4. run sbt assembly 
* 5. Launch hadoop jar
